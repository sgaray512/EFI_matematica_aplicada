# EFI_matematica_aplicada
Para que funcione el archivo .Rmd deben realizar estos pasos en su consola si lo abren desde Linux (ya que estas librerias son del sistema):
sudo apt update

sudo apt install -y \
  libcurl4-openssl-dev \
  libxml2-dev \
  libfontconfig1-dev \
  libfreetype6-dev \
  libpng-dev \
  libtiff5-dev \
  libjpeg-dev \
  libharfbuzz-dev \
  libfribidi-dev

Si lo abren desde Windows no deben instalar nada porque R ya trae estas librerias dentro.
